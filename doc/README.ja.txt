SylSpell プラグイン
================================================================================

このプラグインはSylpheedにスペルチェックを独自に追加するためのサンプルです。

動作環境
--------------------------------------------------------------------------------

* Windows XP SP3/Windows 7 SP1

* Sylpheed 3.1.2以降

インストール方法
--------------------------------------------------------------------------------

1) sylspell.dllをSylpheedのプラグインディレクトリへとコピーします。

2) sylspell.moをロケールディレクトリへとコピーします。

注: Sylpheedをインスーラによりセットアップした場合

%APPDATA%/Sylpheed/plugins以下に配置します。
Windows XPの場合以下の場所にdllをコピーして下さい。
C:\Documents and Settings\(ユーザアカウント)\Application Data\Sylpheed\plugins

使い方
--------------------------------------------------------------------------------

ステータスバーのsylspellのアイコンをクリックして有効にします。
[ツール]-[sylspell]から適宜設定を行います。
